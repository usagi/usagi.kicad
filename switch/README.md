## Cherry MX

- MX Blue と MX Speed Silver の実測値と MX のカタログスペックから当たり判定に大事そうな外形を再現した3Dデータです。
- 端子のホールは実測値に +0.2mm のマージンを加え、パッドはホール +0.65mm を設定してあります。
